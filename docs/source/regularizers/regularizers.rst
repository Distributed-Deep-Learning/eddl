Regularizers
=============

L1
---

L1 regularizer.

Example:

.. code-block:: c++
   :linenos:

    layer L1(layer l,float l1);


L2
---

L2 regularizer.

Example:

.. code-block:: c++
   :linenos:

    layer L2(layer l,float l2);


L1L2
-----

L1L2 regularizer.

Example:

.. code-block:: c++
   :linenos:

    layer L1L2(layer l,float l1,float l2);
